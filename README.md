# IOS Review Application
 
 Gathers the the Most Recent review for an application from the IOS store from the past 48 hours

## Tech Stack
- Vite
- Typescript
- Go
- PostgreSQL
- Docker

## Quick Start
```

// Build application in docker
docker compose build

// Run For first time
docker compose up -d


```
## Technical Decisions

- ViteJS

- Repository Pattern

## Tradeoffs

- Writing to JSON file vs Database
